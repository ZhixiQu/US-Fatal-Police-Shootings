# US Fatal Police Shootings

## Overview

This study analyzes changes in U.S. police shooting fatality rates during the 2015-2024 period. Importantly, this study further examines relationships between victim demographics (e.g., race, gender, age), incident characteristics (e.g., whether armed, threat level, whether fleeing), temporal and spatial factors (e.g., year, state or city), and whether officers activated body cameras.
